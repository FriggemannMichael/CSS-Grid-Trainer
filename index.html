CSS Grid Referenz

        
        

            

                

                    
Container Properties

                    
    
                        
  • display: grid - Erstellt einen Grid Container
    • 
                        
  • grid-template-columns - Definiert Spalten
    • 
                        
  • grid-template-rows - Definiert Zeilen
    • 
                        
  • grid-template-areas - Definiert benannte Grid-Bereiche
    • 
                        
  • gap - Abstand zwischen Grid-Items
    • 
                        
  • justify-items - Horizontale Ausrichtung der Items
    • 
                        
  • align-items - Vertikale Ausrichtung der Items
    • 
                        
  • justify-content - Horizontale Ausrichtung des Grids
    • 
                        
  • align-content - Vertikale Ausrichtung des Grids
    • 
                        
  • grid-auto-columns - Größe automatisch erstellter Spalten
    • 
                        
  • grid-auto-rows - Größe automatisch erstellter Zeilen
    • 
                        
  • grid-auto-flow - Auto-Placement Algorithmus
    • 
                    

                


                

                    
Item Properties

                    
    
                        
  • grid-column-start - Startspalte
    • 
                        
  • grid-column-end - Endspalte
    • 
                        
  • grid-row-start - Startzeile
    • 
                        
  • grid-row-end - Endzeile
    • 
                        
  • grid-column - Kurzform für column-start/end
    • 
                        
  • grid-row - Kurzform für row-start/end
    • 
                        
  • grid-area - Kurzform oder benannter Bereich
    • 
                        
  • justify-self - Horizontale Selbstausrichtung
    • 
                        
  • align-self - Vertikale Selbstausrichtung
    • 
                        
  • place-self - Kurzform für justify-self/align-self
    • 
                    

                


                

                    
Einheiten & Funktionen

                    
    
                        
  • fr - Fraction Unit (flexible Einheit)
    • 
                        
  • repeat() - Wiederholt Muster
    • 
                        
  • minmax() - Min/Max Größe
    • 
                        
  • fit-content() - Passt sich dem Inhalt an
    • 
                        
  • auto-fill - Füllt mit Tracks
    • 
                        
  • auto-fit - Passt Tracks an
    • 
                        
  • span - Überspannt mehrere Tracks
    •
